
# Innate Effects

As the difficulty scales, courses are generated with more and more innate negative effects.

Level 1 effects are given the most, while Level 3 effects only start appearing very late-game.

## Level 1

- [x] Ambiguous - Lectures have a x% chance of not showing their potential understandings.
- [x] Unclear - Lectures have a x% chance of not showing their understand chance.
- [x] Aftershock - After attending attending a lecture about this course, the next lecture (from any course) will take x% more energy.
- [x] Shocked - Attending this lecture takes x% more energy. One time only.
- [x] Travel Cost - Attending this lecture costs $x.
- [x] Takes P - Attending this lecture costs x P.

## Level 2

- [x] Difficult - Lectures appear with an understand chance of no more than x%.
- [x] Unhelpful - Lectures give x% less understanding.
- [x] Exhausting - Attending lectures takes x% more energy.
- [x] Collateral - Attending lectures removes x% of the understanding that you have on every other course.
- [x] Frying - After attending a lecture about this course, the next lecture (from any course) will have an understand chance of x%.

## Level 3

- [x] Bad Pacing - If a lecture has an understand chance below x%, it becomes 0% on attendance.
- [x] Extensive - This course has a goal that is x% higher than normal.
- [x] Mutating - Whenever a lecture about this course appears, lose x% of your current U for this course.
